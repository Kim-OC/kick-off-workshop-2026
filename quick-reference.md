## KiloClaw Commands

```bash
# Start interactive chat
kilo

# One-shot task (agent works autonomously)
kilo run --auto "your task here"

# Useful patterns:
kilo run --auto "Read this and tell me what you find"
kilo run --auto "Analyze this workflow and find inefficiencies"
kilo run --auto "Research how other companies solve [problem]"
kilo run --auto "Design an agent that automates [workflow]"
```

## Lovable Tips

**How to build:**
1. Describe what you want in plain language
2. Watch the app appear
3. Describe changes: "Make the buttons blue" → app updates
4. Keep iterating

**Good descriptions:**
- "Build a kanban board with three columns: To Do, In Progress, Done"
- "Add a form at the top where you can create new tasks"
- "Make each card clickable and show details in a popup"
- "Add a search bar at the top"

**Vague descriptions (avoid these):**
- "Make it look nice"
- "Make it better"
- "Add some stuff"

## Prompt Tips for KiloClaw

**Good prompts:**
- "Read this document and summarize the key points"
- "Find all problems in this workflow description"
- "Research how other companies handle [specific problem]"
- "Design a step-by-step plan for [goal]"

**Vague prompts (avoid these):**
- "Fix this"
- "Make it better"
- "What do you think?"
